# Changelog

## 0.3.0

- Added PLib-based in-game category editor.
- Added custom category ordering in the editor and reroll dropdown.
- Added dynamic interest names from ONI's own `SkillGroup` labels.
- Added English and Brazilian Portuguese UI translations.
- Added safer handling for unavailable interests such as `Rocketry`.
- Added fail-safe behavior so compatibility errors disable the mod for the session instead of crashing the game.
- Added an ONI update watchdog workflow for compatibility reminders.
- Added release packaging script.

## 0.2.0

- Initial public development version.
