# Dupe Interest Sets

Create your own duplicant interest sets and pick them from the reroll dropdown.

This mod lets you add custom categories such as:

- Building + Researching
- Farming + Cooking
- Digging + Suit Wearing + Supplying

Vanilla categories are not changed.

## How to Use

1. Enable the mod.
2. Restart the game if prompted.
3. Open the mod options.
4. Click `Edit Categories`.
5. Add and reorder your custom interest sets.
6. Save and restart.
7. Use the new categories in the duplicant reroll dropdown.

The mod starts with no custom categories. You decide what gets added.

## Notes

- Each category can have 1 to 3 interests.
- Rocketry only appears when the game makes that interest available.
- Traits and other duplicant rolls stay vanilla.
- Includes English and Brazilian Portuguese UI text.
- Requires PLib, bundled with the mod.
- Config changes require a restart.
